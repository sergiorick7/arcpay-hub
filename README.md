# ArcPay Hub 🚀

ArcPay Hub is a decentralized financial application designed to simplify USDC liquidity management and peer-to-peer payments on the Arc Network.

## 🛠 Features
- **Wallet Connection:** Native integration with MetaMask using Ethers.js.
- **Liquidity Management:** Deposit and withdraw USDC directly from your smart contract.
- **Payment System:** Generate dynamic QR Codes for transactions and scan them via camera.
- **Modern Interface:** A clean, professional, and responsive financial UI.

## ⚙️ Tech Stack
- **Network:** Arc Testnet
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Web3 SDK:** Ethers.js v6
- **Utilities:** QRCode.js, Html5-Qrcode
- **Deployment:** Vercel

## ⛓ Contract Information
- **Contract Address:** `0xa5905fD525a823A8ae616aD4C4C51508bA520D27`

## 🚀 How to use
1. Connect your MetaMask wallet.
2. Ensure you are connected to the Arc Testnet.
3. Use the **Deposit** function to add funds to the Hub.
4. Generate a **QR Code** for payments or use the **Camera Scanner** to process incoming transactions.
